# Pro Tech Mobile Website

## Setup Instructions

1. Download all HTML files, `style.css`, and the `assets/logo.png` to a single folder.
2. Replace placeholder text with your shop's real address, phone, email, WhatsApp in `contact.html` (and other pages if desired).
3. For WhatsApp links, use your full number in international format without spaces (e.g., `https://wa.me/923001234567` for Pakistan).
4. To add your map: insert your shop address in the Google Maps embed frame in `contact.html`.
5. Host/upload the folder as a static site (for example on [GitHub Pages](https://pages.github.com)).
6. Update any page content or styling easily. The style is controlled in `style.css`.
